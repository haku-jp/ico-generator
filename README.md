# 🪟 ICO Generator

ブラウザだけでWindowsアイコン（.ico）を生成できる無料ツールです。
サーバー不要・インストール不要・完全クライアントサイド処理。

## 🔗 デモ

👉 [https://haku-jp.github.io/ico-generator](https://haku-jp.github.io/ico-generator)

## ✨ 機能

- PNG / JPG / SVG / WebP に対応
- 8種類のサイズを同時生成（16px〜256px）
- リサイズ方法を選択可能（Cover / Contain / Stretch）
- 透過PNG対応
- 角丸・パディング調整
- リアルタイムプレビュー

## 📐 対応サイズ

| サイズ | 用途 |
|--------|------|
| 16px | タスクバー |
| 24px | 小アイコン |
| 32px | デスクトップ |
| 48px | 中アイコン |
| 64px | 大アイコン |
| 96px | エクストラ |
| 128px | シェル |
| 256px | 高解像度 |

## 🛠 技術スタック

- HTML / CSS / JavaScript
- Canvas API（画像処理）
- ICOバイナリ仕様（純粋なJSで実装）

## 📖 使い方

1. 画像をアップロード（ドラッグ&ドロップ可）
2. 出力サイズを選択
3. 「ICOファイルを生成」をクリック
4. ダウンロード
